Questions

Explain the difference between inline, embedded, and external CSS

Inline css exists within the main section of the html document following a paragraph tag and holds the highest priority, embedded css is held between style tags in the head section of the html document with the second highest priorit, and external CSS uses an external file and links that file through the head section of your html document.

Explain why you should only use external CSS.

We should only use external CSS because it allows for better readability. Also, external CSS allows for others such as graphic designers to edit the design of the website with easy access. Having a separate file via external css can help to create unity throughout the website.

What is a selector used for?

The selector is used to alter specific sections of the html document. For example rather than altering the design of all h1 tags, the selector allows us to direct the desired alterations to specific sections of the html document such as the header section, main section or footer section.
