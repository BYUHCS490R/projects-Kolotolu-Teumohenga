Why is the viewport meta tag important?

The viewport mega tag allows for responsiveness on mobile devices. The viewport mega tag tells your mobile browser not to shrink your webpage down and allows your webpage to remain the normal size

How can you adjust the CSS for printing?

You would create a new css file and when you link it in you will change the media query to fit the needs of which css file to use for viewing the webpage versus which css file to use for printing the webpage.

When would you want to adjust the CSS for printing?

You would only want to adjust the CSS for printing when you would know that someone would want to print out your page.

Why do we need a viewport meta tag?

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

The most recent instruction given to the tag would take priority.

You can create a media tag in CSS that creates a customized screen for widths that are particular to phones or smaller devices.
