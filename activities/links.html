What is the root?

The root is your project folder. It is where your website starts.

What is the relationship between folder and file names and the path?

The domain name matches us to the computer, the computer knows where the folder is and the path will match the files and folders in the root.

What is a default document?

A defualt is a document that a webserver will automatically display when user's visit a website's main address or specific folder without being specific about the file name.

Explain the difference between an absolute link and a relative link. Why would you use relative links?

An absolute link has all the information you need to locate a specific resource on the internet while a relative link will only include a path that is relative to the current page or directory. A relative link does not specify a domain.
