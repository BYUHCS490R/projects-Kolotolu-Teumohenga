When do you use a table?

You will use a table when you need to store data in columns and rows.

When do you not use a table?

You will not use a table when you are laying elements out on your page. It'll make make things tedious.

When do you need to do extra work for table accessibility?

You will need to do extra work for table accessility if you work for a company that can get sued for their website not being accessible to individuals with disabilities.
