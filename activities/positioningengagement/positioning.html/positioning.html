What is normal flow in HTML?

Normal flow is the way the browser lays out the page with block tags and inline tags.

What does float do? Why do you need to clear the float?

Float allows other text to share the space on the page with them horizontally. Float allows text to flow around your content. Clearing the float ensures that text or items that needs to be below the image stays below the image.

What do absolute, fixed, and relative do?

Relative: Keeps the image where it would be in normal flow but it allows you to modify the positioning of the image.
Absolute: This takes your image out of normal flow and doesn't affect other elements.
Fixed: Keeps the picture in place while scrolling

How can you make nice navigation buttons for mobile devices?

You can modify the links in css using the tag “nav a.” Some ways you can modify and make the navigation buttons looks nice include display, padding, background-color, border-bottom and text-decoration.
